# Globaxify Website

Official website for Globaxify – a modern eCommerce store designed to showcase products with a clean UI and smooth user experience.

🌐 Live Website: https://globaxify.online/

---

## About The Project

Globaxify is an eCommerce website built to provide a simple and attractive shopping experience.  
The goal of this project is to create a fast, responsive, and user-friendly store that works smoothly on all devices.

This project focuses on:

- Clean UI design
- Fast loading pages
- Mobile responsive layout
- SEO friendly structure
- Product focused design

---

## Features

- Fully Responsive Design
- Modern UI Layout
- Product Showcase Sections
- Optimized Images
- Smooth Navigation
- SEO Friendly Structure

---

## Tech Stack

This project is built using:

- HTML5
- CSS3
- JavaScript
- Shopify / eCommerce Integration
